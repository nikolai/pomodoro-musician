# Pomodoro Timer

Простой таймер Pomodoro на Python с графическим интерфейсом и звуковыми уведомлениями.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python pomodoro.py
```

## Использование

- Рабочий интервал: 25 минут
- Перерыв: 5 минут
- Длинный перерыв: 15 минут (после 4 рабочих сессий)
- Звуковой сигнал при завершении каждого интервала
